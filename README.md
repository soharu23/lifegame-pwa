# 人生養成遊戲 PWA Closed Beta

這是一個可部署到 Zeabur 的 PWA 靜態網站版本。部署到 HTTPS 後，可以用手機瀏覽器加入主畫面，像 App 一樣開啟。

## 目前功能

- 任務打卡
- 角色頁面，可填寫名字、星座、年齡、tag、自由描述與理想小目標
- 角色頁中的編輯區塊可收合，理想小目標可用清單呈現
- 背包內容整合進角色頁
- 完成任務時有提示音與完成狀態
- 自動累積技能 EXP
- 自動判斷技能等級
- 本周任務板可自由增減顯示任務
- 可重複任務會依日期恢復每日狀態，並顯示本周已達成次數
- 任務可編輯或刪除，刪除不影響過去已累積的 EXP
- 獨立的世界探索 EXP 與世界等級
- 世界探索等級已整理為 Lv.1 到 Lv.10 的稱號、介紹、Buff、道具
- 世界探索升等後解鎖徽章、稱號、Buff、道具
- 不同技能會顯示不同顏色的標籤
- 背包介面將 Buff 和道具分開顯示
- 可自行新增任務
- 可自行新增主線副本
- 主線副本預設為空，可自行新增與刪除
- 主線副本可設定總等分、目前進度，並顯示百分比進度條
- 點擊副本名稱可切換任務詳情
- 手機底部導覽列
- PWA manifest 與離線快取
- 可匯出 / 匯入本機存檔

## Zeabur 部署

1. 將 `life-rpg-pwa-zeabur` 資料夾上傳到 GitHub，或在 Zeabur 選擇 Local Project 上傳。
2. 在 Zeabur 建立 Project。
3. 選 Add Service。
4. 選 GitHub 或 Local Project。
5. 指向這個資料夾並部署。
6. 到 Domains 產生 `.zeabur.app` 網址。

這版不需要資料庫。打卡資料會存在使用者目前瀏覽器的 localStorage。若要保留備份，請到「報告」頁匯出存檔。
